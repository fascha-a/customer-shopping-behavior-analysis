# Customer Shopping Behavior Analysis

## Overview
This repository contains a group data analysis project that examines customer shopping behavior using Python-based data analysis, visualization, and predictive modeling.

The core analysis is built around the `customer_shopping_data.csv` dataset and addresses multiple business-focused questions, including:
- how age and gender relate to spending,
- seasonal and category-level sales patterns,
- which categories generate the highest average transaction value,
- and which payment methods and shopping malls are associated with high-spending customers.

## Repository Contents
- `Data Programming Project.ipynb`: Main notebook with analysis for Questions 1-4, including EDA, visualizations, and models (e.g., Logistic Regression, KNN, SVM, Decision Tree).
- `customer_shopping_data.csv`: Input dataset used throughout the project.
- `Project Proposal.docx`: Initial project proposal.
- `Project Report.docx` / `Project Report.pdf`: Final written report.
- `Group 8 Python Project Presentation.pptx`: Final presentation slides.

## Tools and Libraries
The project is implemented in Python, primarily using:
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## How to Run
1. Ensure Python 3 is installed.
2. Install required libraries (if needed):
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open and run `Data Programming Project.ipynb` in Jupyter Notebook or VS Code.
4. Make sure `customer_shopping_data.csv` is in the same directory as the notebook.
